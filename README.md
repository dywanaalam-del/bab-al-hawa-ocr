# 📄 نظام أتمتة أوامر الصرف المالي - منفذ باب الهوى

تطبيق ذكي مخصص لهواتف الأندرويد والآيفون يقوم بقراءة صور أوامر الصرف المالي باللغة العربية، واستخراج البيانات الأساسية منها تلقائياً، ثم تنظيمها في سجل محفوظات وتصديرها إلى ملفات Excel.

## 🚀 الميزات الرئيسية
* **شاشة بدء تشغيل (Splash Screen):** تحمل شعار الهيئة العامة للمنافذ والجمارك - منفذ باب الهوى.
* **التعرف الضوئي على الحروف (OCR):** معالجة ذكية للنصوص العربية لاستخراج (رقم الأمر، بيان النفقة، المبلغ الصافي).
* **التاريخ التلقائي:** إدراج تاريخ اليوم الحالي تلقائياً في المستند.
* **سجل المحفوظات:** جدول محلي لتجميع كل الأوراق المصورة خلال اليوم.
* **تصدير مجمع:** تحميل السجل بالكامل بضغطة زر واحدة بصيغة Excel (`.xlsx`).
